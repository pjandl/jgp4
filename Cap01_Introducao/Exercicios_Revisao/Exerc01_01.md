# Capítulo 1::Exercício 1

A plataforma é atualmente organizada em dois segmentos:

- JavaSE: é a base de toda a plataforma Java, integra a API considerada padrão e permite o desenvolvimento de aplicações para ambiente desktop ou de médio porte.

- JavaEE: destinado ao desenvolvimento de aplicações complexas e de grande porte, estabelece um modelo composto de múltiplas camadas, adequado para criação de aplicações escaláveis, de grande flexibilidade e confiabilidade.
